日期格式

系统默认能够解析如下日期格式：

yyyyMMdd
yyyy-MM-dd
yyyy/MM/dd
yyyy-MM-dd HH:mm:ss
yyyy-MM-dd HH:mm
yyyy-MM-dd HH
yyyy/MM/dd HH:mm:ss
yyyy/MM/dd HH:mm

除此之外，系统还能将long类型的字符串解析为日期

PageResult基本属性描述

属性名 属性类型 是否可能为null 说明
datas 泛型集合 否 一个不为null，但可能为空的泛型集合，具体请看数据标签描述
totalPage int 否 根据总结果数和每页记录数目计算出的总页码
currentPage int 否 当前页码
listbegin int 否 分页开始页码
listend int 否 分页结束页码
pageSize int 否 每页记录数
totalRow int 否 总记录数
param PageQueryParam 否 校验过后的查询条件，跟原查询条件类型一致，具体请看数据标签描述

一个基本的利用PageResult展示分页栏的模板

<div th:if="${articlePage.totalPage > 1}">
	<ul class="pagination">
		<li
			th:each=" i : ${#numbers.sequence(articlePage.listbegin, articlePage.listend-1)}"
			th:class="${(i == articlePage.currentPage) ? 'active' : ''}"><a
			th:href="${urls.getArticlesUrlHelper().getArticlesUrl(articlePage.param,i)}" th:text="${i}"></a>
	</ul>
</div>

Space属性描述

属性名 属性类型 是否可能为null 说明
name string 否 空间名称
alias string 否 空间别名
createDate Timestamp 否 空间创建日期
isDefault boolean 否 是否是默认空间
isPrivate boolean 否 是否是私人空间
