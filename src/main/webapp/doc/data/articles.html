文章列表标签

基本用法

调用：
<data name="文章列表"/>
<data name="文章列表" currentPage="1"/>
输出结果：
<pre th:text="${articlePage }"></pre>

标签属性

属性名 属性类型 是否必须 默认值 说明
currentPage int 否 1 当前页码
pageSize int 否 10 每页显示的文章数量，取当前pageSize与后台配置的文章pageSize之间的最小值
begin string 否 - 文章最小发布日期，必须配合end属性，该属性值必须能够被解析成日期，点击查看能够被解析成日期的字符串格式
end string 否 - 文章最大发布日期，必须配合begin属性，该属性值必须能够被解析成日期，点击查看能够被解析成日期的字符串格式
query string 否 - 查询标题、标签、别名和内容符合查询条件的文章，超过40个字符会被系统截断(即最长允许40个字符)
tag string 否 - 查询拥有该标签的文章
from enum 否 - 查询复合来源的文章，接收COPIED(转载)以及ORIGINAL(原创)两个值，忽略大小写
sort enum 否 - 排序条件，接收HITS(点击量)以及PUBDATE(发布日期)和LASTMODIFYDATE(最后发布日期)三个值，忽略大小写
highlight boolean 否 true 当query属性值不为空时，是否高亮显示查询结果中复合条件的内容
ignoreLevel boolean 否 false 查询时是否忽略文章置顶
queryLock boolean 否 true 是否查询被锁保护的文章
spaces string 否 - 查询某个或某几个空间下的文章，如果有多个空间，用,分隔（空间别名，非空间ID，如果标签处于某个空间下，那么这个属性不起作用）
queryPrivate boolean 否 只有登录后才能设置该属性，登录后默认为true,否则为false

返回结果

返回一个不为null的PageResult<Article>，点击查看PageResult<?>的属性描述

PageResult的datas在这里为Article，article可供访问属性如下：

属性名 属性类型 是否可能为null 说明
id int 否 文章ID
pubDate Timestamp 否 文章发布日期
lastModifyDate Timestamp 是 文章最后修改日期
title string 否 文章标题
isPrivate boolean 否 是否是私人文章
hits int 否 文章点击数
summary string 否 文章摘要
level Integer 是 文章置顶级别
status enum 否 固定为PUBLISHED
editor enum 否 文章编辑器，值为HTML|MD
from enum 否 文章来源，值为COPIED|ORIGINAL
alias String 是 文章别名
featureImage String 是 文章特征图像
lockId String 是 锁ID
space Space 否 空间
tags Set<Tag> 否 文章拥有的标签，不为null，但可能为空
空间可访问属性如下
属性名 属性类型 是否可能为null 说明
id int 否 空间ID
alias string 否 空间别名
isPrivate boolean 否 是否是私有空间
name string 否 空间名称
lockId String 是 锁ID
Tag可访问属性如下
属性名 属性类型 是否可能为null 说明
name string 否 标签名称

PageResult的param属性在这里为ArticleQueryParam，ArticleQueryParam可访问属性如下

属性名 属性类型 是否可能为null 说明
space Space 是 当前空间点击查看Space的属性描述
begin Date 是 文章最小发布日期，在输出为json格式时，该值为long类型
end Date 是 文章最大发布日期，在输出为json格式时，该值为long类型
query string 是 查询内容，不超过40个字符
status enum 否 固定为PUBLISHED
from enum 是 文章来源，值为COPIED|ORIGINAL中的一个
ignoreLevel boolean 否 是否忽略文章置顶
queryPrivate boolean 否 是否查询私人文章
tag string 是 标签名称
queryLock boolean 否 是否查询锁保护的文章
highlight boolean 否 是否高亮查询结果
sort enum 是 文章排序类型，值为HITS、PUBDATE和LASTMODIFYDATE中的一种
spaces Set<String> 否 指定的空间别名，用于查询一个或多个空间下的文章
spaceIds Set<Integer> 否 指定的空间ID，用于查询一个或多个空间下的文章，由spaces转化
